# Sorting Algorithm Visualizer

An interactive web-based sorting visualizer built using HTML, CSS, and JavaScript.

## Algorithms Implemented
•⁠  ⁠Bubble Sort
•⁠  ⁠Selection Sort
•⁠  ⁠Insertion Sort
•⁠  ⁠Merge Sort
•⁠  ⁠Quick Sort

## Features
•⁠  ⁠Step-by-step visualization
•⁠  ⁠Play / Pause controls
•⁠  ⁠Speed adjustment
•⁠  ⁠Random array generation
•⁠  ⁠Clean UI

## How to Run
Simply open ⁠ index.html ⁠ in any browser.
